# E-Commerce Website
This is a simple E-Commerce Website built using Django during my Full Stack Web Development Internship. The project allows users to browse products, search products, filter products by category, manage a shopping cart, place orders, and perform user authentication.

## Features

- User Login and Registration
- Product Listing
- Product Search
- Category Filtering
- Shopping Cart
- Increase and Decrease Quantity
- Order Placement
- Order History
- Django Admin Panel

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## How to Run

```bash
python manage.py runserver
```

Open the project in your browser:

http://127.0.0.1:8000/

## Project Structure

- EcommerceStore
- store
- media

This project helped me learn Django authentication, session management, product management, shopping cart functionality, order handling, and CRUD operations.
